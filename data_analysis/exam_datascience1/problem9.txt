문제 : 웹에서 데이터를 수집할 때 사용하는 방법은 대표적으로 OpenAPI와 웹 크롤링이 있습니다. 두 방법의 데이터 제공 방식과 수집 방법의 차이점을 중심으로 서술하시오.

- OpenAPI: API키를 이용하여 데이터를 호출하면 이를 데이터 제공 웹사이트에서 데이터 입력으로 반환해주며, API 제공자가 지정한 방식으로 데이터를 제공 받는다. 
- 웹크롤링: 웹페이지를 구성하는 요소(HTML의 id...등)를 데이터 사용자가 직접 선택하고 추출하여 데이터를 '긁어'온다. 
보통 셀레니움과 같이 구글 드라이버를 활용하여 동적인 작동을 하며 자바스크립트 기반 데이터를 수집하는 방법과 뷰티풀숩과 같이 정적인 방식으로 HTML요소 선택자를 활용하여 데이터를 수집하는 방식 등이 있다.
